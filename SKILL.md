# CNKI Research Skill

搜索中国知网 (CNKI) - 中国最大的学术文献数据库，覆盖期刊、学位论文、会议论文、报纸等。

## Capabilities

- **期刊文献检索**: 搜索中文学术期刊论文
- **学位论文检索**: 查找硕士、博士学位论文
- **会议论文检索**: 获取学术会议论文
- **报纸文献检索**: 搜索新闻报道和报刊文章
- **引文分析**: 查看引用和被引用情况
- **学术趋势**: 分析研究热点和发展趋势

## Usage Examples

### 基本检索
```
"搜索 CNKI: 人工智能 医疗"
"查找 CNKI 学位论文: 深度学习 诊断"
"获取 CNKI 会议论文: 心血管 2024"
"搜索 CNKI 期刊: 肿瘤 免疫治疗"
```

### 高级检索
```
"搜索 CNKI: 主题=糖尿病 AND 作者=张三"
"查找 CNKI: 机构=北京大学 AND 时间=2023-2024"
"检索 CNKI 核心期刊论文: 神经网络"
```

### 学术分析
```
"分析 CNKI: 机器学习在医学中的研究趋势"
"查看 CNKI: 某领域的核心作者和机构"
"获取 CNKI: 最新发布的研究热点"
```

## Data Source

- **中国知网** (cnki.net)
- 学术期刊：8,900+ 种
- 学位论文：500+ 万篇
- 会议论文：10,000+ 届
- 报纸：1,000+ 种
- 总文献量：超过 1 亿篇

## Coverage

| 类别 | 内容 |
|------|------|
| 期刊 | 核心期刊、专业期刊、EI 收录 |
| 学位 | 博士论文、硕士论文 |
| 会议 | 国际会议、国内会议 |
| 报纸 | 国家级、省级报纸 |
| 年鉴 | 统计年鉴、行业年鉴 |
| 图书 | 工具书、教材 |

## Notes

- 大多数机构有订阅权限
- 个人用户需要购买访问权限
- 英文论文也有收录
- 提供引文链接和参考文献导出
